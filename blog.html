Discuss the scope of var, let, and const

In sort term, Scope mean where we can access the variable. Basically, scope are two types global scope and functional scope. When we can access the variable anywhere of the document then it is call global scope. And when we can not access the outside of the function where we declare the variable then it is called functional scope variable.

Var:
var is a global scope variable, because we can access the variable from anywhere, inside of the function and outside of the function. Also, var can be redeclare and reassign.

let:
let is a function scope variable, because we can not access the variable out side of a function but if the "let" variable declare outside of a function then it is converted to global scoped. Also, let variable value can be reassign but let variable can not redeclare.

const:
const is also a function scope variable. It's behavior almost same as the let variable, only major difference is const variable can not redeclare and reassign.
